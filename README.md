# ✈️ Sky Routes Profit Dashboard

## 📌 Project Overview

The **Airline Route Profitability Dashboard** is an interactive Business Intelligence project developed using **Power BI** and **SQL**. The project analyzes airline route performance, profitability, revenue, operational cost, seat occupancy, and aircraft utilization. It helps airline management identify profitable routes, monitor flight performance, optimize operational costs, and make data-driven decisions.

This dashboard uses airline operational data from **2022–2026** containing more than **10,000 flight records**.

---

# 🎯 Project Objectives

- Analyze airline route profitability.
- Monitor revenue, operational cost, and total profit.
- Compare aircraft performance.
- Measure seat occupancy across routes.
- Identify top-performing and low-performing routes.
- Track monthly and quarterly profit trends.
- Improve airline business decisions using interactive dashboards.

# 📈 Power BI Dashboard

The dashboard consists of **5 interactive pages**.

---

# 🏠 Page 1 — Overview Dashboard

This page provides a complete summary of airline performance.

### KPI Cards

- Total Flights
- Total Revenue
- Total Operational Cost
- Total Profit
- Average Occupancy %

### Charts

- Top 10 Most Profitable Routes
- Monthly Profit Trend
- Profit by Route Type
- Revenue vs Cost Trend
- Occupancy Gauge

### Purpose

This page gives management an instant overview of airline performance and profitability.

---

# 🌍 Page 2 — Route Performance

This page focuses on route-level performance.

### Visuals

- Route Profitability Map
- Top 10 Routes by Revenue
- Top 10 Routes by Profit Margin
- Route Summary Table

### Purpose

Managers can easily identify profitable routes, compare route performance, and monitor revenue generation across different destinations.

---

# 📉 Page 3 — Trends & Analysis

This page analyzes profit trends over time.

### Visuals

- Monthly Profit Trend
- Profit Trend by Route Type
- Quarterly Profit
- Day of Week Profit Analysis
- Seasonality Heatmap
- Top 5 Loss Making Routes

### Purpose

This page helps identify seasonal demand, monthly growth, weekly performance, and loss-making routes.

---

# ✈️ Page 4 — Aircraft Performance

This page evaluates aircraft efficiency.

### Visuals

- Profit by Aircraft Type
- Aircraft Occupancy Comparison
- Flights by Aircraft Type
- Revenue vs Cost by Aircraft Type

This page helps compare aircraft models based on profitability, occupancy, operational cost, and flight frequency.
